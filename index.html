Product Description

This project is aimed at developing an online Inventory Management System (OIMS) for the IICT of SUST. This system can be used to store the details of the inventory, update the inventory based on the details of different items. This is one integrated system that will be used to request, authenticate and delivery items by designated user classes. Currently at IICT, the store manager(s) is forced to keep track of inventory by hand. This means that they must count what they have bought, allocate and what they have left at the end of each day. They must also fill out order forms to be sent to vendors so that they can restock their inventory in preparation for future. This wastes valuable man hours and is a rather simple task to automate using this software. This OIMS will reduce the human suffering and error substantially from the conventional management system. This Product was developed as academic project of CSE490. Backend was constructed using Django-1.10 and frontend was written with bootstrap and lot of awesome stuffs.
